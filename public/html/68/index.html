釧路の夕焼け

このページの説明

このページは作者の出身地、釧路の夕焼けを画像とコメントで紹介します。

  1. 旧自宅、釧路町遠矢から

    2. 中学時代を過ごした釧路町遠矢、バスケ部だったのでいつも
    夕方になると体育館の窓はピンクから紫がかった夕焼色に染まりました。

  2. 弊舞橋と

    3. 釧路の繁華街近くの橋です。釧路に行った際は食事の前に、
    この景色を見てから行きましょう。

  3. 釧路湿原と

    4. 広大な釧路湿原です。観光の際はタバコ等、火の気には十分
    気をつけましょう。
